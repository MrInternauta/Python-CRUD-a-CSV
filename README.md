# Python registar usuarios en csv
## Descripcion

Registra contactos en una "base de datos" csv

## Instalación
instalar python 2.7

## Uso

```
-Teclea una opción-
        [a]ñadir contacto
        [ac]tualizar contacto
        [b]uscar contacto
        [e]liminar contacto
        [l]istar contactos
        [s]alir

```
